1. Layer

이미지를 불러서 canvas 에 그립니다. 이때 레이어 순서에 맞게 그려주고 가장 밑에는 격자무늬판(checkerboard)을 그려 줍니다.

2. Move Layer

레이어 목록을 표시하고 순서를 변경할 수 있게 합니다. 레이어를 이동시킬 수 있는 컨트롤롤 만듭니다.

    3. Select rectangle

    직사각형으로 선택 기능을 만듭니다. 선택된 패스 도형(path shape; =직사각형)의 외곽선(outline)을 그리는 셰이더(shader)를 만듭니다. 외곽선의 굵기는 2픽셀, 외곽선을 마스크로 하여 왼쪽 위에서 오른쪽 아래로 흐르는 texCoord 기준 기울기가 1인 굵기 2픽셀의 대각선 줄무늬(stripe)를 그립니다.

    4. Select closed path

    패스(path)로 선택 기능을 만듭니다. 선택된 패스 도형(path shape)의 외곽선(outline)을 그리는 셰이더(shader)를 만듭니다. 외곽선의 굵기는 2픽셀, 외곽선을 마스크로 하여 왼쪽 위에서 오른쪽 아래로 흐르는 texCoord 기준 기울기가 1인 굵기 2픽셀의 대각선 줄무늬(stripe)를 그립니다.
